# Telegram Signal Service
A professional market alert bot built for Alfred.

## Setup
1. Open this repo in **GitHub Codespaces**.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Edit `bot.py` and replace `YOUR_BOT_TOKEN_HERE` with your token from BotFather.
4. Run the bot:
   ```bash
   python bot.py
   ```
